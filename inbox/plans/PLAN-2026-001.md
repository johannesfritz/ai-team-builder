## Product Vision

**AI Team Builder** — A visual no-code builder for Claude Code coworker plugins. Users design agentic workflows on a canvas, test them with dry run simulation, and export production-ready plugin packages.

**Current state:** Live at https://jfritz.xyz/ai-team-builder/. Static site on Hetzner. Client-side only with localStorage persistence. No backend, no accounts.

**Next milestone:** Smart Static expansion — shareable URLs, live preview, gallery with fork, GitHub export via Hetzner proxy.

## Risks

| Risk | Severity | Mitigation |
|------|----------|------------|
| Anthropic builds their own builder | Fatal | First-mover speed, deep domain expertise |
| Plugin format changes | High | Abstraction layer in serialize.ts |
| No users (validation failure) | Medium | Plausible analytics to measure, iterate fast |
| localStorage limits (5MB) | Low | Supports ~100 plugins. GitHub export as overflow |
